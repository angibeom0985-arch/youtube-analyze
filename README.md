## 로컬 실행

**필수 사항:** Node.js

1. 의존성 설치:
   ```bash
   npm install
   ```

2. 개발 서버 실행:
   ```bash
   npm run dev
   ```

3. 프로덕션 빌드:
   ```bash
   npm run build
   ```
